# Исследование качества воздуха (SO₂) с использованием машинного обучения

## Описание проекта

В рамках данного проекта было проведено исследование данных :contentReference[oaicite:0]{index=0} (Environmental Protection Agency) за 2025 год, посвящённых уровню диоксида серы (SO₂) в атмосфере.

Основная цель исследования — разработать модель машинного обучения, способную прогнозировать индекс качества воздуха (AQI) на основе географических координат, времени года и региональной принадлежности.

---

## Этапы работы

### Предобработка данных

Для подготовки датасета к обучению были выполнены следующие шаги:

- Из временных меток извлечены дополнительные признаки:
  - `Month` — месяц наблюдения;
  - `Day` — день месяца.
- Пропущенные значения обработаны с использованием метода `SimpleImputer`.
- Категориальный признак, содержащий названия штатов, преобразован в числовой формат при помощи `OrdinalEncoder`.

---

### Построение модели

Для решения задачи регрессии был выбран алгоритм случайного леса — `RandomForestRegressor`.

- Данные были разделены на обучающую и тестовую выборки в соотношении **80/20**.
- Модель обучалась на подготовленных признаках с последующей проверкой на отложенной выборке.

---

### Оценка качества

Для оценки точности модели использовалась метрика **MAE** (*Mean Absolute Error*), которая показывает среднюю абсолютную ошибку предсказания.

---

## Результаты

Построенная модель продемонстрировала среднюю абсолютную ошибку **X.XX** пунктов AQI.

Это означает, что в среднем предсказания модели отклоняются от реальных значений на указанную величину, что свидетельствует о хорошей точности для задачи экологического мониторинга.

## Вывод

Использование алгоритма случайного леса позволило получить качественную модель для прогнозирования AQI на основе пространственно-временных характеристик.

Подобные модели могут быть полезны для:

- мониторинга состояния окружающей среды;
- раннего выявления ухудшения качества воздуха;
- поддержки экологических и городских служб при принятии решений.
